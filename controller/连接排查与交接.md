# eVTOL 接管控制器 — 连接排查结论 & 交接（2026-07-07）

## 一句话结论
控制器**软件已完成、已在 mock 模式端到端验证通过**。没跑通的只有"从这台笔记本连到 sim 主机的 DevKit"，原因是**本机网络环境**（FlClash 代理 + 跨子网 + 本会话无管理员权限），**不是代码问题**。最省事的解法：**把控制器放到 sim 那台机器上本地跑**。

## 已验证 OK
- 控制器 mock：飞到点 → 三类统一断 AP → 风切变速度扰动 → 双通道接管 RT → 落地 → 输出 CSV，全流程通。
- 本机 Python 3.12 可用。
- 本机 → 10.7.144.111:5000 端口层 TCP 可达（但 HTTP 拿不到数据，见下）。

## 连接为什么不通（诊断过程）
1. sim 主机本地 `http://127.0.0.1:5000/get` 正常出 JSON（你确认过，说明 DevKit 本身没问题）。
2. 从本笔记本连 `10.7.144.111:5000`，所有方式都返回 `RemoteDisconnected / 0 字节`：
   - **直连被 FlClash 劫持**：笔记本 Wi-Fi 是 `10.7.160.65/22`，sim 是 `10.7.144.111`，**不在同一子网** → 走默认路由 → 被 **FlClash（TUN 模式，虚拟网卡 198.18.0.1）**抓进代理隧道，到不了局域网主机。
   - 走 FlClash 自带代理端口 `7890`：同样 `RemoteDisconnected`。
3. 想加"只放行 10.7.144.111 走 Wi-Fi 网关"的静态路由来绕过 FlClash → **本会话 PowerShell 非管理员（`IsAdmin: False`），被拒（Access denied）**。UAC 提权需要有人点确认，你离开了没法点。
4. 直连网线：USB 网口已 up（本机 `169.254.217.135`），但对端 sim 无应答（新直连口多被 Windows 归"公用网络"、防火墙挡入站；且无管理员改不了两端静态 IP）。

> 我没有对系统做任何持久改动：加路由都被拒了、FlClash 未改动、网线是物理连接。`config.py` 的 `DEVKIT_BASE_URL` 已设回 `127.0.0.1` 以配合下面方案①。

## 怎么修（任选其一，按省事排序）

### ① 最省事、必成：在 sim 那台机器上本地跑控制器（推荐）
1. 把整个 `controller/` 文件夹拷到 **sim 主机**。
2. `config.py` 里 `DEVKIT_BASE_URL` 保持 `http://127.0.0.1:5000`（已设好）、`USE_MOCK = False`。
3. `python controller.py --check`（打印一帧飞机状态 = 通了）→ `python controller.py`。
4. HMI：sim 主机浏览器开 `http://127.0.0.1:8000/`；别的机器要看就开 `http://<sim主机IP>:8000/`。
5. 彻底没有代理 / 跨子网问题。

### ② 从这台笔记本跑：给终端管理员权限，绕过 FlClash（只放行一个 IP，不动你的代理）
1. **以管理员身份**打开 PowerShell，执行：
   ```
   New-NetRoute -DestinationPrefix "10.7.144.111/32" -InterfaceIndex 19 -NextHop "10.7.163.254" -RouteMetric 1
   ```
   （19 = 你的 Wi-Fi 网卡；10.7.163.254 = Wi-Fi 网关；都是这次实测到的值）
2. `config.py` 的 `DEVKIT_BASE_URL` 改回 `http://10.7.144.111:5000`，`USE_MOCK=False`，`python controller.py --check`。
3. 用完撤销：`Remove-NetRoute -DestinationPrefix "10.7.144.111/32"`。

### ③ 从这台笔记本跑：在 FlClash 里放行局域网
- 打开 FlClash → 网络/TUN 设置 → 开启"**绕过局域网 / Bypass LAN**"，或在规则里加 `IP-CIDR,10.7.144.0/24,DIRECT`（置顶）。
- 然后 `config.py` 指 `http://10.7.144.111:5000`，`--check`。

### ④ 直连网线（彻底避开代理）
- 两端都设**同网段静态 IP**（如笔记本 `192.168.50.2` / sim `192.168.50.1`，掩码 `255.255.255.0`，都需管理员）。
- 把 sim 那块直连网口的网络位置设为"**专用**"，或给 5000 端口放行入站防火墙。
- `config.py` 指 `http://192.168.50.1:5000`，`--check`。
